## Search
```
// mac
"/Applications/Rhino 8.app/Contents/Resources/bin/yak" search  --all --prerelease MyCustomTools
// win
& "D:\Program Files\Rhino 8\System\Yak.exe" search --all --prerelease MyCustomTools
```

## Remove
```
// mac
"/Applications/Rhino 8.app/Contents/Resources/bin/yak" yank MyCustomTools <version>
// win
& "D:\Program Files\Rhino 8\System\Yak.exe" yank MyCustomTools <version>
```
